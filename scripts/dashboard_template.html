What this dashboard is

A reproducible landscape built from curated evidence and conservative ClinicalTrials.gov matches. It is not a disease-incidence map, clinical guideline, or exhaustive regulatory database.

Methods and limitations

This starter deliberately favors traceability over apparent completeness.

Curated evidence

Authorization, WHO-prequalification, stockpile, and programme-use rows are maintained manually in data/curated_candidates.csv with a source URL and verification date.

Automated discovery

A Python script searches ClinicalTrials.gov API v2 using target aliases. It accepts interventional vaccine records, updates matching trial IDs, and never promotes an automated row above stage 5.

Known gaps

Trials in other registries, preclinical programmes, national authorizations, discontinued programmes, and unpublished changes can be absent. Review row-level sources before decisions or publication.
